Connecting to the Glyph

  1. Hold RT2 while plugging the Glyph into USB to enter Configurator mode.
  2. Click Connect in the toolbar and select the Glyph serial port.
  3. Click Load Config to read the current profiles from the device.
  4. Edit your profiles, then click Save to Device to write the new config.

Editing Profiles

  • Click a button on the controller layout to assign or remap it.
  • Use the right panel to edit mode, backends, SOCD rules, and layout.
  • Buttons outlined in teal are active in the profile.
  • Buttons shown as red are disabled (BTN_UNSPECIFIED).
  • Buttons shown with a gold outline are remapped to another button.

Config Files

Use Export JSON / Import JSON to save and restore configs offline.

The JSON format is compatible with the official Glyph Configurator.
